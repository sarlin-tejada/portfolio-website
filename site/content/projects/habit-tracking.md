## The Problem
Daily routines are easy to follow in the moment but difficult to evaluate over time. I wanted to understand how I was actually spending my time and whether it aligned with my intentions and priorities.

## The Data
Google Calendar event data capturing daily activities, time blocks, and routines.

## The Analysis  
- Standardized and categorized events into defined habit groups  
- Extracted raw calendar events using the Google Calendar API  
- Built an ETL pipeline (Python → MongoDB → MySQL) to structure and store data  
- Aggregated activity data into weekly behavior patterns  
- Prepared structured data for visualization and trend analysis  

## Key Insights
- Time allocation revealed gaps between planned routines and actual behavior  
- Some habits remained consistent, while others fluctuated significantly  
- Weekly patterns made it easier to identify where adjustments were needed  
- Small changes in daily routines had a noticeable impact over time
